## Como Restaurar o Backup

1. Extrair o arquivo:
   ```bash
   tar -xzf medsync-source-backup-20250809_184808.tar.gz
   ```

2. Instalar dependências:
   ```bash
   npm install
   ```

3. Configurar banco de dados:
   ```bash
   npm run db:push
   ```

4. Iniciar aplicação:
   ```bash
   npm run dev
   ```
